Data exploration SuRE-SNP project: Overlap with eQTLs, HepG2 vs GTEx Liver

Bas van Steensel, February 2018 - April 2019.

Introduction

Joris has generated SuRE data for millions of SNPs. Here we explore the overlap with GTEx eQTLs: are there any patterns that make biologically and or statistically sense? HepG2 only; see parallel knitr doc for K562.

Description of Data

  • SuRE data are in a dataframe generated by Joris van Arensbergen. This is the full set of SNPs. See location in Joris' folder below.
  • Gtex eQTL significant gene/SNP pair list from all tissues. Downloaded from https://gtexportal.org/home/datasets on 27 Jan 2018, version v7.

#they correctly used the orientation of the gene to infer which end of the gene is the TSS;
#but they forgot that the sign of the distance should also be flipped if orientation=="-"
#as provided, if orientation=="-", then tss_distance<0 means that the SNP is *downstream* of the TSS
#biologically this is not right, so let's calculate my own distance column:
Q$snp2tss<-ifelse(Q$orientation=="-", Q$end-Q$snp_pos, Q$snp_pos-Q$start)

##Focus on TSS-proximal SNPs:
#These data indicate that concordance is higher near TSSs. This is not surprising, and also in agreement with other papers such as Degner et al 2012. 
#So how does concordance depend on distance cutoff? 
DisCut<- 10^seq(3, 6, by=0.1)
#DisCut<- c(0, 1e3, 1e4, 1e5, 1e6)
for(i in 1:length(DisCut))
{T1<-table(DQ$concordant[ abs(DQ$snp2tss)<DisCut[i]])
T2<-table(DQ$concordant[DQ$SNP_ID %in% Dhits & abs(DQ$snp2tss)<DisCut[i]])
#significance:
M<-matrix(c(T1-T2, T2), ncol=2)
Tst<-fisher.test(M)
if(i==1){ConcorDF<-data.frame(distance.cutoff=DisCut[i], all=sum(T1), hits=sum(T2), OR=Tst$estimate, pvalue=Tst$p.value)}
else {ConcorDF<-rbind(ConcorDF, data.frame(distance.cutoff=DisCut[i], all=sum(T1), hits=sum(T2), OR=Tst$estimate, pvalue=Tst$p.value))}
}

#list  ORs:
print(ConcorDF)
